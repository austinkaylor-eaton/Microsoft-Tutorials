# Microsoft-Tutorials
Examples of tutorials found and followed on [Microsoft Learn](https://learn.microsoft.com/)

## New Project Steps
1. Create a new solution folder with the title of the project name
2. Run the command `mkdir "name-of-project"` in a terminal
3. cd into the new project folder
4. Run the command `dotnet new sln` in a terminal
5. Add any new project files to the solution file inside of the project directory
6. Follow the tutorial
